# Típus

## Struktúra

```
public class Type
 {
     [Key] // This attribute specifies that Id is the primary key
     public int Id { get; set; }

     [Required] // This attribute specifies that Name is required
     public string Name { get; set; }
     public string Img { get; set; }
 }
```
Ez a kód a **Type** nevű osztályt definiálja, amely egy típus entitást reprezentál.


## A legfontosabb elemek:

- **Id**: Egyedi azonosító *(primary key)*, amely egyedi módon azonosítja a típust az adatbázisban.

- **Name**: A típus neve, kötelező mező.

- **Img**: A típushoz tartozó kép URL-je.

Ez az osztály tehát a típusok adatainak tárolására szolgál, a játékban, és tartalmazza a típus nevét és a hozzá tartozó képet.
