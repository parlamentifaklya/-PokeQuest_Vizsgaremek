# Eszközök

 Használt programok és nyelvek

## Programok
 
**Web frontend**:
- Visual Studio Code

<details>
  <summary>Miért pont Visual Studio Code? </summary>
- A Visual Studio Code garantálja a megfelelő kényelmet és struktúrát a programozáshoz.
- Jó verziókezelő UI-a van.
- Lightweight
- Könnyen bővíthető add-on / pluginokkal.
</details>

**Web backend, Admin alkalmazás**:
- Visual Studio 2022

<details>
  <summary>Miért pont Visual Studio? </summary>
- A fentebbiekben megemlített pozitívumok ugyanúgy itt is megtalálhatóak.
- Kódolás közben segítséget nyújt / tippeket ad.
</details>
