# Rendszer követelmények

- A játék futtatása és élvezhetősége érdekében, érdemes egy legalább 1920x1080 pixel felbontású full HD monitor.
- Körülbelül 1GB tárhely a letöltéshez.
- Javasolt, hogy a játékhoz egy stabil internetkapcsolat álljon rendelkezésre.
- Ezen kívül semmi, hiszen böngészőben futtatható.
