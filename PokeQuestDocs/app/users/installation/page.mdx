# Telepítés

## Elsődleges lépések

Három dolgot kell kötelezően letölteni a program működéséhez:

- [Git](https://git-scm.com/downloads): Ez kell a repository klónozásához.
- [Node.js](https://nodejs.org/en/download): Ez a frontend (weboldal) futtatásához kell.
- [.NET 9.0](https://download.visualstudio.microsoft.com/download/pr/b0032fde-aac9-4c3e-b78c-4bd605910241/8d2aa21baac4aef9b996671cd8a48fb2/dotnet-sdk-9.0.202-win-x64.exe):
 Ebben a .NET verzióban lett írva a backend (szerver) ezért kell ez a kifejezett verzió a futtatáshoz.

## Ha meg vannak az Elsődleges lépések

1. Nyiss meg egy parancssort (cmd-t)

2. Klónozd le a repot a következő parancshasználatával
```
git clone https://github.com/Ordinary56/Findgroup.gg.git
```
3. Nyisd meg a klónozott mappát. Vagy manuálisan fájlkezelőből, vagy parancssorban a következő paranccsal:
```
cd <A mappa neve amibe lépni szeretnél>
```

4. Amikor bent vagy a Findgroup.gg mappában akkor `Web` mappában lesz a frontend és a backend mappa
>[!NOTE]
>
> A most következő a parancsokat is parancssorban kell kiadni az adott mappában, vagyis nyiss meg egy parancssort a mappában ha fájlkezelőből nyitottad meg a mappát.

### Backend 

A következő parancsokat használd a backend megfelő elíndításához amikor a backend mappában vagy

1. Migráld az adatbázist .
```
dotnet tool install --global dotnet-ef
dotnet-ef migrations add <Itt add meg a migráció nevét>
dotnet-ef database update
```
2. Fordíts le a programot.
```
dotnet build
```
3. Futtasd a programot.
```
dotnet run
```

### Frontend

A következő parancsokat használd a frontend megfelő elíndításához amikor a frontend mappában vagy

1. Telepítsd a függőségeket a következő paranccsal:
```
npm install
``` 
>[!NOTE]
>
> Ha nem jönne le az összes függőség akkor ezeket is futtasd le:
```
npm install @mui/material @emotion/react @emotion/styled
npm install clsx
npm install axios
```
2. Futtasd le a programot a következő paranccsal:
```
npm run dev
``` 
3. Ez után nyisd meg a terminálban megjelenő localhost linket

```
http://localhost:5173
```
